# Getting Started

First of all, you must copy it to your project:

### `git clone git@github.com:conservativ007/momentum.git ./<your_directory>`

Then install the dependencies:

### `npm install`

# Available Scripts

In the project directory, you can run:

### `npm run start`

Runs the app in the development mode.\
Open [http://localhost:8080](http://localhost:8080) to view it in the browser.\
Enable React Hot Reloading.

### `npm run build`

Builds the app in the development mode to the `build` folder.

### `npm run build-prod`

Builds the app for production to the `build` folder.

### `npm run clean`

Delete `dist` folder.

#### Technology stack:
##### `html, css, js, react, webpack`
